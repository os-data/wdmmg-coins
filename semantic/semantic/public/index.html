The pages served by this host support autonegotiation of content-types. If you look at them with a web browser, it will request them as text/html and web pages will be returned. However if you look at them with an RDF browser or request them from a computer program, the Accept header should be set to application/rdf+xml in which case the RDF serialisation will be returned.

The SPARQL endpoint does not yet support autonegotiation but this is planned for the near future.
